## 1일 차
과정소개

학습 리포지토리 생성

- 마크다운 학습
1. 기본문법 - 제목
    ```markdown
    # 제목1
    ## 제목2
    ### 제목3
    #### 제목4
    ##### 제목5
    ###### 제목6 (잘사용안함)
    <!-- 주석(HTML주석 동일) -->
    ```

2. 기본문법 - 목록
  ```markdown
  - 목록
  * 목록
  1. 숫자목록
  2. 숫자목록
  ```

3. 기분문법 - 링크, 이미지
    ```markdown
    [네이버](http://naver.com)
    
    ![이미지](이미지URL)
    
    ## 사이즈 조절 이미지
    src : 이미지URL
    width : 이미지 넓이 픽셀단위 지정
    <img src="이미지URL" width="500">
    ```
